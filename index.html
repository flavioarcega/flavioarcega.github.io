Visão Geral

Dar uma Visão Geral sobre o tema.

Problemas comuns

  • Perda de código-fonte
  • Falta de controle de acesso aos artefatos
  • Ausência de gestão das mudanças e atividades
  • Mudanças que reintroduzem erros
  • Impossibilidade de recuperar determinado estado/versão do sistema
  • Falta de sincronismo entre as atividades
  • Alterações paralelas e conflitantes que se sobrescrevem
  • Grande dificuldade em integrar as partes do software
  • Ambientes de desenvolvimento e produção diferentes (na minha máquina funciona)

Conceitos Associados

  • Artefato

    Refere-se a todos os produtos de trabalho gerados durante o desenvolvimento de software (artefato = produto de trabalho).

  • Produto de Software

    Refere-se ao subconjunto de todos os artefatos gerados no projeto que é entregue ao cliente e/ou usuário final. (executáveis, documentação, utilitários de instalação, etc)
